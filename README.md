# Copy Pasta - A simple clipboard manager for windows
## Features

- Add fields in the application
- Edit fields in the application
- Remove fields in the application
- Copy fields to clipboard
- Saves data and loads up on launch
-----------------

## How to use

- Add fields in the application
- Copy fields to clipboard
